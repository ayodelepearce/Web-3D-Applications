5. 3D App Deeper Understanding Statement with feedback from testing

Deeper Understanding: This project implements a PHP PDO database abstraction layer serving a RESTful API, hardware-aware rendering using window.devicePixelRatio, anisotropic texture filtering, sRGB colour space correction, MVC architecture, and resolution of complex Blender GLTF export issues including texture node incompatibility and modifier baking.

Testing Feedback: Tested on M1 MacBook Air using Google Chrome and Safari via PHP built-in server (localhost:8000) and deployed to the Sussex ITS server. All three models load correctly, all interactive buttons function as expected including wireframe, lighting, rotate and reset view.
